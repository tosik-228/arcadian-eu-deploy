# Arcadian EU Deploy

Minimal public deployment artifact for the Arcadian landing page on DigitalOcean App Platform.

This repository intentionally contains only:

- static HTML/CSS/JS/assets for the public landing page;
- the DigitalOcean Function wrapper for the contact form;
- the DigitalOcean App Platform spec template.

Secrets are configured in DigitalOcean App Platform environment variables and are not stored here.
